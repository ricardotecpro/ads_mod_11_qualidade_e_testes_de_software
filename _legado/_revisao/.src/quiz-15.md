# Quiz 15 - Publicação e Google Play Store 🚀

1. O que é o arquivo `.aab` (Android App Bundle)?
    - [ ] Um áudio em alta qualidade
    - [x] O formato moderno de publicação que otimiza o tamanho do app para cada celular
    - [ ] O código fonte do Android comprimido
    - [ ] Um tipo de antivírus do Google
    *Explicação: O AAB substituiu o APK como padrão de envio para a loja desde 2021.*

2. Quanto custa para criar uma conta de desenvolvedor no Google Play Store?
    - [ ] Grátis para sempre
    - [ ] $99 por ano
    - [x] Taxa única de $25
    - [ ] 10% de cada venda do app
    *Explicação: Diferente da Apple, o Google cobra apenas uma vez para você ser dev oficial.*

3. Para que serve a `Chave de Assinatura` (Keystore / .jks)?
    - [ ] Para abrir o prédio do Google
    - [x] Para garantir que o app é seu e impedir que outras pessoas enviem atualizações maliciosas
    - [ ] Para criptografar as fotos do usuário
    - [ ] Para ligar o modo desenvolvedor no celular
    *Explicação: É a identidade digital única do seu aplicativo.*

4. O que acontece se você perder a Chave de Assinatura (`.jks`) de um app publicado?
    - [ ] Você pode pedir uma nova por e-mail no dia seguinte
    - [x] Você nunca mais poderá atualizar o app; terá que criar um novo do zero com outro nome
    - [ ] O Google apaga seu app da loja
    - [ ] Você terá que pagar a taxa de $25 de novo
    *Explicação: A segurança da chave é responsabilidade TOTAL do desenvolvedor.*

5. O que o `ProGuard / R8` faz com o código?
    - [ ] Traduz o código para outras línguas
    - [x] Ofusca (embaralha) e remove código desnecessário para diminuir o tamanho e aumentar a segurança
    - [ ] Corrige bugs automaticamente
    - [ ] Adiciona propagandas no app
    *Explicação: Ele torna o app menor e mais difícil de ser "hackeado" ou copiado.*

6. Qual o objetivo principal dos "Canais de Teste" (Interno, Beta, Fechado)?
    - [ ] Ganhar dinheiro antes do lançamento
    - [x] Validar o app com poucos usuários reais antes de abrir para todo o mundo
    - [ ] Divulgar o app para a imprensa
    - [ ] Testar se a conexão com o Wi-Fi é rápida
    *Explicação: Permite corrigir falhas graves sem afetar a nota do app na loja pública.*

7. O que são os "Recursos da Loja" (Store Assets)?
    - [ ] Os computadores que rodam o app
    - [x] Ícones, Descrições e Capturas de Tela (Screenshots) que convencem o usuário a baixar o app
    - [ ] Os servidores da Google
    - [ ] As moedas virtuais compradas dentro do app
    *Explicação: É a vitrine do seu trabalho para os usuários.*

8. Como o iOS (Apple) distribui versões de teste para os desenvolvedores?
    - [ ] Via WhatsApp
    - [x] Através do app TestFlight
    - [ ] Mandando um pendrive pelo correio
    - [ ] Não é possível testar antes de publicar
    *Explicação: O TestFlight é a ferramenta oficial e centralizada da Apple para testes beta.*

9. O que é o `Google Play Console`?
    - [ ] Um novo videogame do Google
    - [x] O portal onde o desenvolvedor gerencia seus apps, vê estatísticas e responde avaliações
    - [ ] O local onde o código Kotlin é escrito
    - [ ] O nome do suporte técnico do Android
    *Explicação: É o painel de controle administrativo de quem publica apps.*

10. Quanto tempo leva, em média, a primeira revisão de um app pelo Google?
    - [ ] 5 minutos
    - [x] De 1 a 7 dias (dependendo da complexidade e histórico do dev)
    - [ ] Sempre 1 mês exato
    - [ ] O Google não revisa apps, eles entram na hora
    *Explicação: Existe uma análise de segurança e política antes do app ficar visível para o público.*
