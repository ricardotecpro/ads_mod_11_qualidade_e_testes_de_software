# Exercícios: Aula 09 - Tipos de Testes Funcionais ✅

## 🟢 Básico

1.  **Smoke Test**: Por que este tipo de teste é chamado de "Teste de Fumaça"? Qual o seu principal objetivo ao receber um novo build?
2.  **Regressão**: Em qual momento do ciclo de vida você deve rodar os testes de regressão? 

## 🟡 Intermediário

3.  **Aceitação (UAT)**: Imagine que os desenvolvedores dizem que o software está pronto, mas o cliente diz que "não era bem isso o que eu pedi". Onde o processo de teste de aceitação falhou?
4.  **Exploratórios**: Cite 3 características de um bom testador exploratório. Como ele se diferencia de um testador que segue apenas scripts pré-definidos?

## 🔴 Desafio

5.  **Análise de Impacto**: Uma alteração foi feita apenas na tela de "Pagamentos". Quais módulos você incluiria em um teste de **Regressão parcial**, além do próprio módulo de pagamentos? Justifique.
