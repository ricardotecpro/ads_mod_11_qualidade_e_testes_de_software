# Quiz 12 - UX/UI e Material Design 3 🎨

1. O que é o `Material Design`?
    - [ ] Uma ferramenta para criar sites em HTML
    - [x] O sistema de design do Google que fornece diretrizes estéticas e funcionais para apps
    - [ ] O nome da tela de vidro do smartphone
    - [ ] Um tipo de plástico biodegradável usado em celulares
    *Explicação: O Material Design ajuda a criar apps consistentes, bonitos e acessíveis.*

2. Qual a principal novidade do `Material 3` (Material You)?
    - [ ] O app agora fala com o usuário por voz
    - [x] Cores Dinâmicas que se adaptam ao papel de parede do usuário
    - [ ] O fim do uso de botões
    - [ ] Todos os apps agora devem ser obrigatoriamente amarelos
    *Explicação: O sistema extrai a paleta de cores do wallpaper para personalizar a interface do app.*

3. Para que serve o `FAB` (Floating Action Button)?
    - [ ] Para fechar o app rapidamente
    - [x] Representar a ação principal e mais importante de uma tela (ex: +, Enviar, Ligar)
    - [ ] Mudar o brilho da tela
    - [ ] Indicar que o celular está carregando
    *Explicação: É aquele botão circular que fica "flutuando" no canto inferior da tela.*

4. Qual a diferença visual entre o Material Design e o iOS (High Interface Guidelines)?
    - [ ] O Android não usa cores
    - [x] Material foca em elevação, sombras e hierarquia clara; iOS foca em transparências e minimalismo
    - [ ] No iOS os botões não funcionam
    - [ ] O Material Design foi criado pela Apple
    *Explicação: Cada sistema tem sua própria "filosofia" visual e padrões de interação.*

5. O que são `Vector Assets` e por que usá-los?
    - [ ] São fotos pesadas que ocupam muita memória
    - [x] São ícones baseados em fórmulas matemáticas (SVG) que não perdem qualidade ao ampliar
    - [ ] São códigos secretos do Android
    - [ ] São sons de notificação em alta fidelidade
    *Explicação: Vetores garantem que o ícone fique nítido em qualquer resolução de tela e economizam espaço no app.*

6. Por que o "Modo Escuro" (Dark Mode) é importante?
    - [ ] Para o usuário não conseguir enxergar o app à noite
    - [x] Economia de bateria (em telas OLED) e maior conforto visual para o usuário
    - [ ] Para esconder os erros de design do programador
    - [ ] Porque a cor preta é mais barata que a branca
    *Explicação: Telas modernas apagam os pixels pretos, o que economiza muita energia.*

7. Como o Android Studio ajuda a organizar cores para o modo dia e noite?
    - [ ] Ele muda a cor do código automaticamente
    - [x] Usando pastas diferentes: values/colors.xml (dia) e values-night/colors.xml (noite)
    - [ ] O programador deve mudar o código manualmente toda vez que o sol se põe
    - [ ] Ele tira uma foto do céu para decidir a cor
    *Explicação: O sistema troca automaticamente os recursos com base na configuração do celular.*

8. O que é o efeito `Ripple`?
    - [ ] Um erro que faz a tela tremer
    - [x] É o feedback visual de "onda" que aparece quando o usuário toca em um item clicável
    - [ ] O som que o app faz ao abrir
    - [ ] Um tipo de transição de tela que imita água
    *Explicação: O ripple confirma para o usuário que o app recebeu o comando do toque.*

9. Qual a importância das margens e espaçamentos (Paddings)?
    - [ ] Nenhuma, quanto mais informação junta, melhor
    - [x] Facilitar a leitura, evitar cliques errados e dar um ar "premium" e organizado ao app
    - [ ] Gastar mais tinta na tela do celular
    - [ ] Esconder os botões do usuário
    *Explicação: O "respiro" no design é essencial para uma boa experiência de uso (UX).*

10. O que significa "Acessibilidade" no contexto de UI/UX?
    - [ ] Ter um app que custa barato
    - [x] Garantir que pessoas com diferentes habilidades (visuais, motoras, etc) consigam usar o app
    - [ ] Ter um botão que abre a porta de casa
    - [ ] O app funcionar sem precisar de internet
    *Explicação: Inclui contraste de cores adequado, suporte a leitores de tela e tamanhos de fonte ajustáveis.*
