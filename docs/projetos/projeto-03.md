# Projeto 03: Canvas do STLC 🔄

## 🎯 Objetivo
Estruturar o Ciclo de Vida de Testes para uma funcionalidade nova.

## 🛠️ O que fazer
Funcionalidade: **"Login via Biometria no App Bancário"**.
1.  **Análise**: Liste 3 dúvidas que você enviaria para o PO (Dono do Produto) sobre este requisito.
2.  **Planejamento**: Defina qual será o ambiente de teste necessário (Simuladores? Aparelhos reais?).
3.  **Desenvolvimento**: Escreva o título de 3 Casos de Teste (1 positivo, 1 negativo, 1 de borda).

## 📤 Entrega
Um mapa mental ou lista estruturada seguindo as fases do STLC aplicadas ao login por biometria.
