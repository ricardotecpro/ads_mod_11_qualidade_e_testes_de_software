# Projeto 11: TDD na Prática 🔴🟢🔵

## 🎯 Objetivo
Exercitar o pensamento Red-Green-Refactor.

## 🛠️ O que fazer
Requisito: Criar um validador de senhas.
- Regra: A senha deve ter no mínimo 8 caracteres.
1.  **Fase RED**: Escreva como seria a chamada do teste unitário passando uma senha de 5 caracteres. Qual o erro esperado?
2.  **Fase GREEN**: Escreva a função mínima para fazer esse teste passar.
3.  **Fase REFACTOR**: Como você melhoraria o código se agora tivesse que validar também a presença de um número?

## 📤 Entrega
A sequência de pseudocódigo ou código (Python/JS) demonstrando os 3 passos do ciclo.
