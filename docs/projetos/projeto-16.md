# Projeto 16: Portfólio de QA 🎓

## 🎯 Objetivo
Consolidar todas as entregas do curso em um único repositório de portfólio.

## 🛠️ O que fazer
1.  Crie um arquivo `README.md` principal para o seu portfólio de estudos.
2.  Organize os links para os mini-projetos mais importantes que você realizou (ex: Projeto 04, Projeto 11, Projeto 14).
3.  Escreva um parágrafo sobre qual área de QA você mais se identifica (Manual, Automação, Performance ou Segurança) e por quê.

## 📤 Entrega
O link (ou o conteúdo) do seu README de portfólio consolidando sua jornada no curso.
