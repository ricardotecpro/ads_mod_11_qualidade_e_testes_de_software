# Setup do Ambiente: VS Code e Extensões 💻

O Visual Studio Code é o editor de texto mais utilizado por QAs devido à sua flexibilidade.

## 1. Instalação
Baixe em [code.visualstudio.com](https://code.visualstudio.com/).

## 2. Extensões Recomendadas para QA
Abra o menu de extensões (`Ctrl+Shift+X`) e instale:

- **Python**: Suporte completo para a linguagem e Pytest.
- **Playwright Test for VSCode**: Permite rodar e debugar testes web com um clique.
- **Thunder Client**: Alternativa leve ao Postman dentro do VS Code.
- **Error Lens**: Destaca erros de sintaxe diretamente na linha do código.
- **GitLens**: Visualização avançada do histórico do Git.
- **Mermaid Editor**: Para visualizar os diagramas de testes que criamos.

## 3. Dica de Produtividade: Atalhos
- `Ctrl + Shift + P`: Abre a paleta de comandos.
- `Ctrl + ' `: Abre o terminal integrado.
- `F5`: Inicia o Debug.
