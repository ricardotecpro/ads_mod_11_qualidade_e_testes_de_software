# Técnicas de Caixa Branca ⚪
## Aula 08 - Testes Estruturais

---

## O que é Caixa Branca? 🔍

- Foco na lógica e estrutura do código
- Requer acesso ao código-fonte
- Geralmente feito por desenvolvedores (Testes Unitários)

---

## Cobertura de Instruções 📑
### Statement Coverage

- Garante que cada linha de código foi executada pelo menos uma vez
- Ajuda a achar o "código morto"

---

## Cobertura de Decisões ⚖️

- Garante que cada ramo do `IF` foi testado (Verdadeiro e Falso)
- Mais robusto que a cobertura de instruções simples

---

## Complexidade Ciclomática 🤯

- Mede o número de caminhos independentes
- Quanto maior a complexidade, mais difícil é testar
- Funções complexas precisam ser refatoradas!

---

## Cobertura de Fluxo de Dados 🌊

- Rastreia como as variáveis são usadas e modificadas
- Encontra erros de inicialização de variáveis
