# Gestão de Defeitos 🐛
## Aula 15 - Bug Life Cycle e Jira

---

## O Ciclo do Bug 🔄

1.  **Novo**: Reportado
2.  **Aberto**: Validado pelo QA
3.  **Resolvido**: Fixado pelo Dev
4.  **Reteste**: QA verifica a correção
5.  **Fechado**: Problema encerrado!

---

## Um Bom Report de Bug 📝

- **Título**: Curto e direto
- **Passos**: 1, 2, 3...
- **Resultado Atual**: O erro
- **Resultado Esperado**: O correto
- **Evidências**: Prints e Logs

---

## Severidade vs Prioridade ⚖️

### Severidade
- Impacto técnico no software

### Prioridade
- Urgência de negócio / entrega

---

## Gestão Ágil com Jira 📋

- Kanban Boards para fluxo visual
- Backlog de bugs
- Dashboards com métricas de qualidade

---

## Bug Triage 🤝

- Reunião entre QA, Dev e PO
- Decidir o que corrigir primeiro
- Limpeza do backlog
