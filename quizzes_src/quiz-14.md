# Quiz 14 - Testes de API 📡

1. O que significa a sigla API?
   - [ ] Advanced Python Interface.
   - [x] Application Programming Interface.
   - [ ] Automated Project Integration.
   - [ ] Authorized Personal Information.
   > **Explicação**: APIs permitem a integração e comunicação entre diferentes sistemas.

2. Qual o protocolo de comunicação mais comum para APIs web modernas?
   - [ ] Bluetooth.
   - [ ] FTP.
   - [x] HTTP / HTTPS.
   - [ ] SMS.
   > **Explicação**: O protocolo HTTP é a base para APIs RESTful.

3. Qual método HTTP é usado para CRIAR um novo recurso no servidor?
   - [ ] GET.
   - [x] POST.
   - [ ] DELETE.
   - [ ] HEAD.
   > **Explicação**: POST é o verbo padrão para criação de dados.

4. Se uma API retornar o status "201 Created", isso significa:
   - [ ] Houve um erro de servidor.
   - [x] A requisição foi bem-sucedida e um novo recurso foi criado.
   - [ ] O usuário não tem permissão.
   - [ ] O recurso não foi encontrado.
   > **Explicação**: Códigos da família 2xx indicam sucesso.

5. O que indica a família de status codes 4xx?
   - [ ] Sucesso absoluto.
   - [ ] Erro de infraestrutura de rede.
   - [x] Erro do Cliente (ex: dados inválidos ou falta de permissão).
   - [ ] Redirecionamento de página.
   > **Explicação**: 4xx indica que o problema está na requisição enviada.

6. O Postman é uma ferramenta utilizada primariamente para:
   - [ ] Editar vídeos.
   - [x] Desenvolver, testar e documentar requisições de API.
   - [ ] Formatar o computador.
   - [ ] Jogar online.
   > **Explicação**: Postman é a ferramenta líder de mercado para interação com APIs.

7. Em uma requisição de API, onde geralmente enviamos o Token de Autenticação?
   - [ ] No corpo (Body) do email.
   - [x] No Cabeçalho (Header) da requisição, geralmente como "Authorization".
   - [ ] Na URL do site oficial.
   - [ ] Não é necessário enviar tokens.
   > **Explicação**: Headers transportam metadados essenciais, incluindo credenciais.

8. O formato de dados mais popular para troca de informações em APIs REST é:
   - [ ] XML.
   - [x] JSON (JavaScript Object Notation).
   - [ ] CSV.
   - [ ] TXT.
   > **Explicação**: JSON é leve, fácil de ler e amplamente suportado.

9. Qual a principal vantagem de testar a API antes da Interface (Frontend)?
   - [ ] É mais colorido.
   - [x] Permite isolar bugs de lógica de negócio e as respostas são muito mais rápidas que o carregamento de telas.
   - [ ] Não precisa de internet.
   - [ ] O desenvolvedor não fica sabendo.
   > **Explicação**: Testar a API garante que o "motor" do sistema está funcionando corretamente.

10. O status "500 Internal Server Error" indica que:
    - [ ] O usuário digitou a senha errada.
    - [x] Ocorreu uma falha inesperada no processamento dentro do servidor.
    - [ ] A página foi movida para sempre.
    - [ ] O teste passou com louvor.
    > **Explicação**: 5xx indica problemas do lado do servidor/backend.
